# siscatalogo
SisCatalogo es un robusto sistema web construido en PHP moderno, optimizado para gestionar catálogos de productos, controlar existencias e inventarios, y generar o recibir cotizaciones estructuradas y pedidos en línea desde dispositivos móviles y de escritorio. El sistema incluye un Catálogo Público Inteligente y un Panel de Administración avanzado
